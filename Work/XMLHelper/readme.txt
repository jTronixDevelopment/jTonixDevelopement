To Run

DownLoad Node
https://nodejs.org/en/

Copy and paste folder anywhere on you computer.

Use command line to navigate to the directory.

Type:

  npm install http-server -g

Next type:

  http-server -p [What ever Port you choose]
  *** if left of the port defaults to 8080



First click the start program.
To use Copy and paste the XML you are trying to search to the ATT.xml file.
This will give you the best accuracy.
Then filter result after searching.
